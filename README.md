# HexaGo server

## Setup environment

```
# Install bundler
gem install bundler

# Install gems
bundle install
```

## Start server

```
rails s
```

## Swagger documentation

Swagger is used to document our API, you can access the current documentation at the following links:

- **JSON doc**: http://localhost:3000/swagger_doc.json
- **UI doc**: http://localhost:3000/swagger
